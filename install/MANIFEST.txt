PACKAGE CREATION:
-----------------

To create the installation package:

1. Build the mod:
   gradle clean build

2. Copy JAR to install directory:
   cp build/libs/ai-minecraft-player-*.jar install/

3. Verify all files are present:
   ls -R install/

4. Create archive:
   cd install/
   zip -r ../ai-minecraft-player-install.zip *
   # Or:
   tar -czf ../ai-minecraft-player-install.tar.gz *

5. Test installation:
   - Extract package in a test location
   - Run quick-install.sh
   - Verify all files copied correctly
